Lenguaje de programacion

- Palabras reservadas: var, let, const, if, for, function...
- Operadores: +, -, *, /, %, // &&, !, typeof

  - Host (browser, NodeJS)
      - objeto Windows -> BOM -> alert(), setTimeout()
      - Objeto Documento -> DOM
      - Nuevos APIs de HTML5 -> video
      - Objetos global de node -> setTimeout()
